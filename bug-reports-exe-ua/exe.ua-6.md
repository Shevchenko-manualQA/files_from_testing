## [EXE-6] No error message when uploading a document instead of a user avatar 

**Severity:** Sev-2  
**Priority:** Medium 

### Environment
- Windows 10, Google Chrome 123.0.6
- Android 13

### Preconditions
- The user is on the profile page and is about to upload a photo.

### Steps
1.	Click on the “Завантажити зображення”(Upload image) button.
2.	Select a document file (e.g., .pdf, .docx) instead of an image.
3.	Click the “Завантажити“ або “Зберегти“(Upload or Save) button.

### Expected Result
An error message appears informing the user that the selected file is not a valid image.

### Actual Result
No error message is shown. 

**Type of bug:**
Functional
